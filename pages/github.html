PWEB1

Breve descrição: E-commerce simples em Java com Spring Boot e JPA/Hibernate para API REST, persistindo dados em PostgreSQL, consumida por um front-end separado.

Tecnologias utilizadas: Spring Boot, JPA/Hibernate, PostgreSQL, Node.js, SASS, Gulp.

Justificativas: Spring Boot reduz configuração e inclui Tomcat embutido; Hibernate simplifica mapeamento ORM e persistência; PostgreSQL é gratuito e confiável; Node.js oferece flexibilidade e leveza no ambiente de build; SASS facilita manutenção de estilos e Gulp automatiza tarefas repetitivas.

PDMI-VamosRachar

Breve descrição: App Android em Java (Android Studio) que facilita o rateio de contas entre várias pessoas, incluindo cálculo de taxa de serviço e funcionalidade de compartilhamento.

Tecnologias utilizadas: Java, Android Studio, ConstraintLayout, Material Design Components, Intent API.

Justificativas: Java oferece performance e compatibilidade ampla; Android Studio acelera o desenvolvimento com ferramentas integradas; ConstraintLayout possibilita layouts responsivos em diferentes tamanhos de tela; Material Design assegura uma interface consistente; e Intent API permite compartilhamento nativo de resultados de forma simples.
